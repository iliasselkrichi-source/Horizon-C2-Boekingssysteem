                sequenceDiagram
                    participant User as 👤 Gebruiker
                    participant Form as 📝 Formulier
                    participant JS as ⚡ JavaScript
                    participant API as 📡 Supabase API

                    Note over User,Form: 1. GEBRUIKER VULT FORMULIER IN
                    User->>Form: Kiest unit/pakket + datums

                    Note over User,JS: 2. KLIK OP "BEVESTIG RESERVATIE"
                    User->>JS: Klikt op knop

                    Note over JS: 3. JSON FICHE WORDT SAMENGESTELD
                    JS->>JS: {team, unit_code, resource_id, gast_naam, gast_contact, check_in, check_out, metadata, totaalprijs}

                    Note over JS,API: 4. 📤 VERZENDING NAAR SUPABASE
                    JS->>API: POST /rest/v1/boekingen (zonder boekingsnummer!)

                    Note over API,DB: 5. DATABASE ONTVANGT DATA
                    API->>DB: INSERT INTO boekingen

                    Note over DB,Trig: 6. TRIGGERS WORDEN UITGEVOERD
                    DB->>Trig: check_resource_availability() + generate_booking_number()

                    Note over DB,RT: 7. REALTIME BROADCAST
                    DB->>RT: Nieuwe boeking wordt gepubliceerd

                    Note over RT,Panel: 8. REALTIME LISTENER ONTVANGT UPDATE
                    RT->>Panel: postgres_changes (INSERT)
                    RT->>Agenda: postgres_changes (INSERT)

                    Note over Panel,Agenda: 9. UI WORDT LIVE BIJGEWERKT
                    Panel->>Panel: refreshUI() - tabel update
                    Agenda->>Agenda: refreshUI() - kalender update

                    Note over JS,User: 10. FEEDBACK
                    JS-->>User: Alert met boekingsnummer
